    Read count bytes from the input stream. Does not return until at least count bytes have been read. If the end-of-file is reached or an I/O error occurs before the required number of bytes have been read then an exception is thrown.

    Throws

    an exception if the end-of-file is reached or an I/O error occurs before the required number of bytes has been read.
