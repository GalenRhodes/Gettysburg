  • parseKeyValueData(data:)

    Parses out a string of Key/Value pairs. This method scans the input string for key/value pairs in the format <whitespace>key="value". An apostrophe can be used in place of the double quotation mark but, which ever one is used, it has to end with the same one it starts with. In other words you can’t have key="value'. Also, spaces around the equal sign (=) is allowed.

    Throws

    if the input was not is the proper format.
