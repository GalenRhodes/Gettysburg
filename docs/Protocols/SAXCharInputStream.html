SAXCharInputStream

public protocol SAXCharInputStream : CharInputStream

SAXParser version of the CharInputStream protocol that add a number of features needed by the SAXParser.

    Read an XML Name from this character input stream. This method reads characters until the first non-XML Name Character, such as whitespace, is encountered.

    Throws

    if there is an I/O error or if errorOnEOF is true and the EOF is encountered.

    Declaration

    Swift

    func readXmlName(mustHave: Bool = true, errorOnEOF: Bool = true, leadingWS: LeadingWhitespace = .Allowed) throws -> String

    Parameters

    mustHave

    if true then at least the starting character must be read and the returned string will not be empty. If false the returned string might be empty.
    errorOnEOF

    if true and the EOF is encountered then an error is thrown. If false and the EOF is encountered then the characters read so far are returned.
    leadingWS

    tells if any leading whitespace is allowed, required, or not allowed.

    the string of characters.
